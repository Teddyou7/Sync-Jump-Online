# 👬 同步跳跃 (Sync Jump Online) - 哥俩好 Web版

这是一个基于 HTML5 Canvas 和 Python FastAPI (WebSocket) 开发的双人合作平台跳跃游戏。两位玩家被一根具有真实物理弹性的绳子连接在一起，需要默契配合，通过蓄力跳跃在随机生成的平台上不断向前攀登。一人失足，另一人可以依靠地形作为“锚点”将其拉拔起飞！

## ✨ 游戏特色

- **双人联机 / 本地同屏**：支持共用同一键盘的本地双人模式，以及基于 WebSocket 的极速在线匹配、开房联机模式。
- **硬核物理绳索**：基于 Verlet 积分（渲染）和胡克定律（物理）实现的弹性绳索系统。绳索会在达到最大长度时产生强制约束。
- **绝境救援机制**：队友掉下悬崖？只要你不掉下去，你就是拥有“无限质量”的安全锚点！此时原地蓄力起跳，能利用猛烈的拉力把悬在半空的队友强行“拔拔”上来。
- **蓄力跳跃**：长按跳跃键进行形变蓄力，松开即可根据蓄力时间产生不同高度和距离的跳跃，生动考验两人的微操与节奏感。
- **随机生成地形**：程序化生成的无限关卡，包含普通平台、超级弹力平台（Bounce）、踩踏崩塌平台（Collapse），难度随时间动态提升。

## 🛠️ 技术栈

- **前端**：原生 JavaScript (Vanilla JS) + HTML5 Canvas API + CSS3
- **后端**：Python 3.9+ + FastAPI + Uvicorn
- **网络通信**：WebSocket（以服务端为权威的物理运算，保障多端状态严格同步）

## 🎮 玩法指引

### 操作方式
*   **本地模式 (Local Co-op)**：
    *   **玩家 A (蓝方)**：按住并松开 `A` 键 / `左方向键` 蓄力跳跃。
    *   **玩家 B (红方)**：按住并松开 `D` 键 / `右方向键` 蓄力跳跃。
*   **联机模式 (Online Multiplayer)**：
    *   **本机玩家**：按住并松开 `空格键 (Space)` / `W键` / `向上方向键` 蓄力跳跃。
    *   *注：将左上角生成的房间码分享给朋友即可加入，或清空输入框点击直接匹配。*

### 生存法则
1. 你的跳跃距离和高度取决于 **按住按键的时间**（画面中小人的身体会被压缩，上方会出现进度条，最大有效蓄力为 1.2 秒）。
2. 不要同时跃向未知的悬崖。始终保留一个人在稳定的平台上，以便在另一个人失误落空时作为“安全锚点”。
3. 两人都在空中跌落，并且低于所有可视平台一定距离时，游戏才会结束（Game Over）。

## 🚀 本地开发与运行指南

### 1. 克隆代码
```bash
git clone https://github.com/your-username/sync-jump-online.git
cd sync-jump-online
```

### 2. 安装环境依赖
请确保你的电脑上安装了 Python 3.9 或更高版本。
```bash
pip install fastapi uvicorn
```

### 3. 启动服务端
进入项目根目录，运行以下命令启动服务器：
```bash
python server/main.py
```
默认会在控制台输出：`Uvicorn running on http://0.0.0.0:8000 (Press CTRL+C to quit)`。

### 4. 开始游戏
打开你的现代浏览器（Chrome / Edge / Firefox 等），访问：
👉 `http://localhost:8000`
